# Lista de Exercícios – Funções em Java

## Programação de Soluções Computacionais

### Sobre a atividade

Esta lista tem como objetivo praticar a utilização de funções em Java, aplicando conceitos fundamentais de programação como modularização, passagem de parâmetros, retorno de valores, estruturas de repetição e manipulação de dados.

---

## 📋 Exercícios

### 1. Impressão de padrão numérico

Geração de um padrão em que cada linha contém a repetição do número correspondente à linha.

### 2. Sequência numérica crescente

Impressão de uma sequência crescente iniciando em 1 até o número da linha atual.

### 3. Soma de três valores

Implementação de uma função responsável por receber três argumentos e retornar sua soma.

### 4. Verificação de sinal

Função que retorna:

* **'P'** para números positivos;
* **'N'** para números negativos ou iguais a zero.

### 5. Cálculo de imposto

Função `somaImposto` utilizada para calcular o valor final de um produto após a aplicação de uma taxa de imposto.

### 6. Conversão de horário

Conversão do formato **24 horas** para o formato **12 horas (A.M./P.M.)**, permitindo múltiplas consultas.
